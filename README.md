# 📡 Telco Customer Churn Prediction System

An AI-powered web application that predicts whether a telecom customer is likely to leave the company (customer churn) using Machine Learning.

This project is built using Python, Streamlit, Scikit-learn, Pandas, and Plotly with an interactive dashboard and batch prediction system.


## 🚀 Features

- 📊 Interactive Analytics Dashboard
- 👤 Single Customer Churn Prediction
- 📂 Batch Customer Risk Processing
- 🎨 Multiple UI Themes
- 📈 Interactive Visualizations using Plotly
- 🤖 Machine Learning Model Integration
- 📥 Export High-Risk Customers as CSV
- ⚡ Fast and Responsive Streamlit Interface
